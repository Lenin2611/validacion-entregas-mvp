# validacion-entregas-mvp

## Lenin Muñoz & Mary Salcedo

El MVP implementado automatiza la validación del formato del archivo al momento de realizar un push en el repositorio. Mediante GitHub Actions, se ejecuta un workflow que detecta los archivos subidos y verifica si su extensión corresponde a formato PDF. En caso contrario, el sistema genera un mensaje de error y detiene la ejecución del flujo. Si el archivo cumple el formato requerido, el sistema confirma la validación exitosa.

Este prototipo demuestra la automatización de una parte clave del proceso TO-BE, permitiendo validar requisitos técnicos de manera automática y reduciendo la intervención manual del docente.
